# Todo List App

A simple command-line todo application built with TypeScript and Node.js.

## Features to Implement

- **Add an (U)pdate command** to edit existing todos
- **Add a priority level** (high/medium/low) to each todo
- **Add a "clear all" command** to delete everything at once
- **Show a count** of how many todos you have

## Requirements

- Node.js (v14 or higher)
- npm

## Installation

```bash
npm install -D @types/node
```

## Running the App

```bash
npx tsc todoApp.ts
node todoApp.js
```

## How to Use

Commands: `add`, `list`, `remove`, `clear all`, `exit`

## Technologies Used

- TypeScript
- Node.js readline module
- ES Modules
